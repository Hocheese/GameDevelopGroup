字段名 类型 说明

表sys_profession

二级学院与专业列表
id int 系统标识符
name text 专业/二级学院名
college int 二级学院。sys_profession中的id

表teams_info

团队基本信息表
id int 系统标识符
tname text 队伍名
tcreator int 创建者
timeline int 创建时间

表teams_user

团队用户表
tid int 团队id
uid int 用户id

表tips

小贴士表
id int 系统标识符
tips text tip
timeline int 添加时间

表user

用户信息表
id int 系统标识符
account_code int 6位数账号
password text 加密过的密码
email text 邮箱
admin int 管理员等级.0为普通用户
level int 用户等级（预留）
realname text 姓名
sex int 性别。0为无，1为男，2为女，3为其它
study_code int 学号
profession int 专业。表sys_profession中的id
timeline int 注册时间

表user_punchin_month

本月用户签到表
ucode int 用户账号
date int 表示本月的一个日期。0——31之间
ispunch int 值为0,1。表示本月是否签到。当月份为奇数时，1表示已经签到。当月份为偶数时，0表示已经签到。(当date mod 2=ispunch时表示已经签到)

表user_punchin_times

各月用户签到次数表
ucode int 用户账号
year int 年份
month int 月份
times int 签到次数
